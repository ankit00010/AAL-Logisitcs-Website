   ```bash
   git clone https://github.com/ankit00010/AAL-Logisitcs-Website

   ```

   - Backend:
     ```bash
     cd backend
     npm install
     ```
   - Frontend:
     ```bash
     cd frontend
     npm install
     ```

   - Create a `.env` file in the root directory and add the necessary credentials (e.g., database URI, mail service credentials, JWT secret).

     ```bash
     npm run server
     ```

     ```bash
     npm start
     ```
